# 固定指令

顶部200px
左边300px

<template lang="pug">
.v-xxx
  div(v-pin="200") 顶部200px
  div(v-pin:left="300") 左边300px
</template>
<script>
  export default {
    name: 'v-pin'
  }
</script>

# 说明

v-pin 支持传入定位位置和定位值v-pin:left || top 默认 left

# API

Name Type Description Required
arg String left||top ➖
value String||number 0 ➖
